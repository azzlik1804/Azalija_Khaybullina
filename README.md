 Описание
Графическое приложение на Python для генерации случайных задач. 
Приложение позволяет фильтровать задачи по категориям, добавлять свои собственные задачи и сохраняет историю сгенерированных заданий в файл.

 Функционал
1. **Генерация**: Случайный выбор задачи из списка.
2. **Фильтрация**: Возможность выбрать категорию (Учёба, Спорт, Работа) или смотреть все.
3. **Добавление**: Пользователь может добавить свою задачу, выбрав категорию из списка.
4. **История**: Все сгенерированные задачи сохраняются в `tasks.json` и отображаются в списке.
5. **Валидация**: 
   - Нельзя добавить пустую задачу.
   - Категория выбирается строго из предустановленного списка (защита от опечаток).
   - Корректная обработка пустого или поврежденного файла истории.

 Требования
- Python 3.6+
- Стандартные библиотеки: `tkinter`, `json`, `random`, `os` (не требуют установки).
